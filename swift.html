Data is distributed over a set of machines, to keep track of where they reside. Swift does this by maintaining internal data structures called “Ring”.

Partition

  • device的映射，一个或多个partition会对应于一个device
  • Number of partitions = 2^part_power
  • device的增减不会影响partition的总数，只需要改变它们之间的映射，这保证了存储结点不会因为设备的增减而剧烈抖动
因此，为了将来的扩展，partition的数量需要设定大一些

The Ring Data

The Ring Data structure consists of three top level fields:

  • List of Devices，集群中设备的列表
  • Partition Shift Value，计算hash的位移量
  • Partition Assignment List，partition到device的指派, a list of lists

Building the Ring

  • Calculates the number of partitions to assign to each device based on the weight of the device. For example, for a partition at the power of 20, the ring has 1,048,576 partitions. One thousand devices of equal weight will each want 1,048.576 partitions.
  • Assign each partition replica to the device as far away as possible from other replicas
For example, for a partition at the power of 20, the ring has 1,048,576 partitions. One thousand devices of equal weight will each want 1,048.576 partitions.

Building the Ring

  • When rebuilding a ring, recalculates the desired partitions that each device wants
  • Gather partitions for reassignment, include from removed devices, any that can be spread out for better durability, and random partitions from any devices that have more partitions than they need
  • Reassign the gathered partitions to devices by the similar method
  • No partition will be moved twice in a configurable amount of time

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

Custom Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

Slide Backgrounds

Set data-background="#007777" on a slide to change the full page background to the given color. All CSS color formats are supported.
